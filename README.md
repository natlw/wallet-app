# Wallet - Agregator Portfela Kryptowalut i Walut

**Zintegrowany System Informatyczny służący do monitorowania portfela inwestycyjnego użytkownika w oparciu o aktualne dane rynkowe.**

## Funkcjonalności:
- Rejestracja i logowanie użytkowników
- Zarządzanie aktywami (CRUD: dodawanie, wyświetlanie, edycja, usuwanie)
- Automatyczne pobieranie kursów z zewnętrznych API (CoinGecko / NBP)
- Inteligentne buforowanie danych (Cache) chroniące przed limitami zapytań (Rate Limiting)
